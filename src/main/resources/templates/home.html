Surveys - это сервис, который предоставляет возможность создавать, управлять и проходить опросы онлайн. Пользователи могут создавать разнообразные опросы, определять вопросы и ответы, экспортировать статистику результатов в машиночитаемом формате JSON.

Платформа позволяет удобно пройти опросы - Surveys обеспечивает простой и удобный интерфейс как для создателей опросов, так и для участников, делая процесс создания, управления и прохождения опросов эффективным и интуитивно понятным.

Surveys - это платформа, где участники получают вознаграждения за прохождение опросов. Пользователи могут заработать условную валюту, выполняя определенное количество опросов. Эти вознаграждения могут быть обменяны на деньги в соотношении 1 к 1, которые можно вывести на банковскую карту.

Создание опросов на платформе Surveys предоставляется за плату. Пользователи, желающие создать свой опрос, могут пополнить свой баланс в зависимости количества опросов, которые они планируют разместить на платформе. Пользователи могут либо только проходить опросы, либо только создавать.

Ниже представлен наш интерактивный конструктор вопросов, а именно типы вопросов, которые может создать пользователь.

Создание опроса

Есть несколько типов вопросов соответственно кнопкам. В каждом вопросе можно добавить несколько изображений. Ниже представлено описание кнопок и типов вопросов.

  • Добавляет вопрос с одним текстовым вариантом ответа
  • Добавляет вопрос с одним числовым вариантом ответа
  • Добавляет вопрос с несколькими вариантами ответа из списка - чекбоксами
  • Добавляет вопрос с одним вариантом ответа из списка - радио
  • Добавляет вопрос с одним вариантом ответа - датой
